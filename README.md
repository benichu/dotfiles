# Ben's dotfiles

## Installation

```bash
git clone https://github.com/benichu/dotfiles.git && cd dotfiles && ./bootstrap.sh
```

To update, `cd` into your local `dotfiles` repository and then:

```bash
./bootstrap.sh
```
