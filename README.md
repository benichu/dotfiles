# Ben's dotfiles

## Installation

```bash
git clone https://github.com/benichu/dotfiles.git && cd dotfiles && ./bootstrap.sh
```

## Update

```bash
cd ~/dotfiles && git pull https://github.com/benichu/dotfiles.git && ./bootstrap.sh
```
